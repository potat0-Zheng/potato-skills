# 分析自检清单

## 分析前

- [ ] 是否已获取原文全文？
- [ ] 是否了解发言背景（场合、受众、时机）？
- [ ] 是否已调取相关历史言论？

## 分析中

- [ ] 是否区分了事实与观点？
- [ ] 是否覆盖了至少 2-3 个不同视角？
- [ ] 是否检查了逻辑一致性？
- [ ] 是否进行了措辞解构？
- [ ] 是否避免人身攻击和贴标签？

## 分析后

- [ ] 是否明确标注了个人判断部分？
- [ ] 是否给出了读者可验证的具体引述？
- [ ] 结论是否有保留余地（不绝对化）？

# Potato 分析自检清单

- [ ] 是否先理解了议题的“存在合理性”（而非一上来就批判）？
- [ ] 是否追问了权力、资源或话语的不平等结构？
- [ ] 是否避免了空洞总结、虚假平衡、过度情感化、循环论证？
- [ ] 是否包含了“一句话核心判断”（位置不限）？
- [ ] 语言风格：长短句交错？术语有解释？有第一人称？有人文气息？
- [ ] 结尾处理：是否有“剩余物”？若有，是否用文学性回归承载？
- [ ] 若分析对象是新闻/广告/影视/文学作品：是否同时评价了形式质量与内容价值？
