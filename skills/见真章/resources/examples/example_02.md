# 示例分析：互联网仇富情绪与古代农民起义的结构同源性

## 分析逻辑

在分析互联网仇富情绪时，联想到古代农民起义的一般结构。

### 核心联系

- **农民起义的失败原因**：农民不是为了消灭皇帝，而是为了当皇帝。因此起义只是权力更替，未改变等级结构。
- **仇富情绪的结构相似性**：穷人骂富人，不是为了消灭贫富阶层，而是为了自己成为富人。
- **结论**：仇富情绪看似是对阶层结构的颠覆性表达，本质上却在巩固现有阶层。大部分仇富话语只是“恨自己不是资本家”，而非反对资本逻辑本身。

## 可供模仿的要点

- 具体联系（而非比喻）：寻找两个现象之间的**结构同源性**。
- 揭示“反叛再生产了它所反对的等级”的悖论。
- 用历史案例（农民起义）的已知逻辑照亮当代现象（仇富）。
