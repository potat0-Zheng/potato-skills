# 培养文科思维
对理科思维的强调实际上构建了文理对立的二元框架，无论理科思维是否客观存在，当它被提及的那一刻，文科思维也不得不存在了。
按这个逻辑来说，如果文科思维只是作为“被贬抑”的对象而存在，那未免有点太凄惨了。因此，此文需要以另一种独立的、完整的定义为文科思维申辩。
如果社会上对理科思维的普遍感知或定义是：一种逻辑、实证、系统分析的思考方式，那么它的核心不是“数据”“逻辑”，而是“可证伪性”——即当一个命题或理论，只有存在被推翻的可能性时，才属于科学。依照这一结论，文科思维的核心应该是：可诠释性。它是一个在对某个议题去意识形态基础上完成再意识形态化的过程。

对理科思维的褒扬是启蒙思想的延续，彼时的理性是对神权与王权的挑战。理性的地位被无限放大，并在神权与王权消失的此时仍占据着思想权力的主导地位。这一论述在现代生活中很容易得到例证：几乎所有领域，人们总是更能够相信数据结果而非经验结果。20世纪20年代，Beech-Nut培根公司为推销自家的培根，找来了名盛一时的公关大师伯内斯为其制作营销计划。伯内斯通过问卷的形式向5000名医生发放问题：你认为丰盛早餐好还是清淡早餐好。结果是有近4500名医生选择了前者。医生作为“科学”“严谨”“实验”的绝对代言人，民中国对理科思维的崇拜自然延续到了这一结论上，早餐也在培根和麦片的推广下成为了一天不可或缺的环节。
在中国的具体语境下，现代学术基础几乎复制了西方的研究范式，无论在科学技术领域还是人文社科领域，以数据和实证为核心的方法论始终占据重要地位。因此，中国传统中以实践经验、直觉感知的方法被抛弃，甚至被唾弃。两者轻而易举地形成了一个十分简单的二元对立框架，而理科思维也在21世纪，这一生物的世纪、化学的世纪、计算机的世纪、AI的世纪，唯独不是文科的世纪的不断实践中被强化，推上高位。

这便引到了本文的一个批判重点：如果理科思维是去直觉的、可被证伪的，那么它就意味着绝对客观、绝对安全。也就是说，理科思维指向了一个去意识形态化的思考。
事实是否真的如此？大家都接受一千个读者有一千个哈姆雷特，但是不接受公共议题有除了我以外的其他解读，也不接受不同语境下类似结果的不同处理。这种自我认知下的“去意识形态”是危险的，因为客观在日常使用语境中拒绝了灰色空间，拒绝了多重结果。
就像你有两个朋友吵架了，争执不下之时找你前来：“来，你说句公道话。”此题难解之处在于：双方都是你的至爱亲朋、手足兄弟，一旦在气头上指出某一方的不对，TA便可能不可置信地看着你，声泪俱下地同你不再往来。此时你也不得不无奈摇头，手搭在双方的肩膀上说一句：“我客观地说，你俩都有问题”。
但是要是另一方是你的仇敌，那事情便变得明朗许多——你扬了扬手，挑了挑眉，说：“我客观地说，就是你的问题”。
你与朋友或仇敌的关系可以被视作微缩的意识形态，而在这种事情发生时，就算在明面上的变量一致，在做出最终判断时，你仍然会将一些隐藏变量纳入考虑，比如双方的为人，当时场景是公共场合还是私人空间。因此，明面上做出的“客观”最终不过是做了伪装的偏袒罢了。
在这一案例中，人们很快能够意识到这一点，但是在对于更宏大的议题时，这种“去意识形态”便成了一种傲慢与偏见。人们往往热衷于在社会议题中寻找可证伪性，但没有人能在一些难以通约甚至难以定义的领域（自由与安全，公平与效率等）中做一个庞大的社会实验，但人们依然热衷之，热衷复刻所有制度因素复现北欧福利，复刻所有文化变量实现文娱昌盛。

因此，有些问题不能够被充分解释，只能被诠释。所谓文科思维的“可诠释性”本质上是对某事某物的定义。我们可以清楚地说出DNA的定义，全世界人都不会对此有异议。但什么是自由，什么是民主，这些词汇的定义权是抢来的。在理科思维的框架下，这些问题有一个绝对的对与错，这是因为理科思维是在一个既定范式下思考，而文科思维需要你亲手选择一个范式。
所以在文科思维里，我们需要充分意识到，针对人文社科的思考，去意识形态的操作是不现实的，不是无法实现，而是没有必要。如果一个对社会或个体的讨论不最终落到对这个个人和社会的反思或修正或改良，那么讨论就是无意义的，是空洞的。如果讨论成立，那么立场与意识形态必然存在。
因此，再意识形态化的过程是必要的。需要认识到，选择什么问题、采取什么方法、认定什么有效的过程本身就是意识形态的产物。与其以客观之名掩之，不如大方承认。因为李普曼说过：当你形成一个观点时，也许你对事件的真实情况远谈不上了解。而这一警示本身就能帮助我们避免一些认知上的谬误。
