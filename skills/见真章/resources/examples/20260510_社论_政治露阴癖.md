# 政治露阴癖的分析
有相当一群人在抖音平台上，在内容创作与内容互动中会急不可耐地用一些特定的符号展现自己的政治立场。比如在一个国外小孩在国外草地玩耍的视频底下，高赞的评论往往是“文明的模样👍”。
不过到现在，随着“斩杀线”的概念爆火，“鉴证”之风越吹越大。“土殖”与“粉红”攻守异形，只要在科技、民生、文化等广泛领域出现一点“对比”的意味，评论区的几拨人便闻着味来到战场进行魔法对轰——即使视频内容是一只可爱狗狗。部分人将这种现象称为：政治露阴癖。
政治露阴癖这里指代在互联网场域中，将自己的政治主张通过显式或隐式的方式在作品或评论的表达出的冲动。这种表达与心理学上的露阴癖有象征意义上的共通。第一，政治主张在人际交流中属于较为隐私的部分，这同理于人体的生理部位；第二，露出的场合往往违背一般期望，特别是在公共场域，且常出现在与政治完全不相关的领域；第三，他们在露出中获得一定的快感；第四，面向的对象的限定——露阴癖的对象是自己的性幻想对象，而政治露阴癖往往通过狗哨政治的表达形式对特别人群进行表达。
