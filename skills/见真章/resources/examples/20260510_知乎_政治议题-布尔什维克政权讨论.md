# 如何看待布尔什维克在选举失败后动用武力拒绝承认选举？



在施密特的理论体系中，“合法性”与“正当性”不仅不是一回事，而且往往处于对立状态。自由主义的悲剧就在于试图用“合法性”来吞噬“正当性”。所谓“合法性”就是只要符合现有的法律条文、经过了议会半数通过、履行了行政程序，就是“合法”的。合法性是功能性的。它像是一台机器，谁掌握了51%的选票，谁就能启动这台机器。施密特指出，如果只讲合法性，那么当敌人利用法律程序来摧毁法律本身时（例如像纳粹那样通过合法选举上台并废除民主），法治国家将无力还击。而正当性的最高来源是人民作为政治统一体的存在。一个国家的宪法之所以有效，不是因为它逻辑严密，而是因为背后有一个具有“政治生命”的主体（人民）做出了决断。立宪权在宪法产生后并不会消失，它只是“隐遁”了。当国家秩序崩溃时，立宪权会再次显现，撕毁旧契约，创造新现实。自由主义法治观认为法律可以规范一切，但施密特指出，法律无法预见所有的极端危险。当国家面临生死存亡的“例外状态”时，现有的法律规范会失效。在法治的真空地带，谁能站出来说“现在是例外状态，法律暂时停止运营”，谁就是真正的主权者。主权者并不存在于法律条文之中，而是存在于法律的边界之上。施密特认为人民在政治中的地位，等同于上帝在神学中的地位。上帝通过“神迹”介入自然规律，主权者则通过“决断”介入法律秩序。主权者的决断之所以正当，是因为他代表了人民想要“作为这个特定国家存续下去”的根本意志。作为整体的“人民”是一个巨大的、混沌的、平时并不显现的政治实体。人民无法直接管理税务、签署条约或指挥军队。主权者的任务，就是通过“决断”让模糊的人民意志变得“清晰可见”。自由主义认为议员是人民的“代理人”（像律师）；而施米特认为主权者是人民的“象征”或“道成肉身”（像耶稣）。主权者在例外状态下的“决断”，不是在替人民办事，而是在那一刻，他就是人民意志的肉身化。这里要强调的是主权者是耶稣不是上帝，在政治神学上这个区分的意义在于那种“非完全神性”的、作为中介的剩余部分。如果主权者是上帝，“主权者即人民”，他天然正确、不会犯错、不可质疑；如果是耶稣，“主权者使人民显现”，则代表行为始终存在再现的失败风险。人民这个“混沌的巨大实体”永远比主权者的任何一次决断都多出一些东西，永远有被更好地代表、被更真切地召喚的可能。这就保留了一个最低限度的神圣性跃迁。当人民呼唤时，“象征”是可以被替换的。既然主权者是人民意志的“体现”，那么主权者所做的决断，即便违反了成文法律，也都可以辩称是为了实现“更高的人民意志”。当敌人利用合法性摧毁合法性，这时主权者应该以“正当性”来废除合法性。不过施密特强调的是，在魏玛宪法的框架下，总统作为“宪法守护者”，其正当性恰恰来源于宪法本身赋予的紧急权力。这是一种“宪法之内的例外”，即以宪法的某一核心原则，去悬置那些次要的条文程序。施密特的理论不是为了给革命正当性，而是为了更好地保卫秩序。​一种秩序之所以正当，是因为它体现了某种具体的价值取向和政治生活方式。​决断的正当性在于它重建了秩序。施密特有一句名言：“就像所有的法律一样，例外状态也需要秩序。”决断不是为了制造混乱，而是为了终结混乱，这种“恢复秩序”的能力本身就是一种正当性。​当主权者站出来划分敌友、做出决断时，如果人民承认“这就是我们的意志”，那么这一行动就获得了超越法律条文的正当性。当一个政权的合法性变成了摧毁其正当性的工具时，正当性必须挺身而出，通过决断来废除这种已经僵死的合法性。解散立宪会议的正当性在于，立宪会议无法解决迫在眉睫的政治危机，或者说它本身已经成为了混乱的源头。如果布尔什维克被认为比议会更能体现“人民”这一政治主体的真实意志，那么它的行动就具有正当性。布尔什维克的武力既是在破坏合法性，也是在通过一种“创始性的暴力”，去建立一套全新的合法性。立宪会议代表的是“规范”，而革命代表的是“例外”。规范只能在和平时期运作，而在生死存亡的斗争中，谁能实施有效的决断，谁就是真正的主权者。即主权者就是那个决定例外状态的人。立宪会议是一个追求“代表性”（各党派、各阶级通过议员代理）和“理性讨论”的机构。它代表了“规范”。布尔什维克（自命的）宣称自己不是部分利益的“代理人”，而是“无产阶级”和“人民”这个政治统一体本身的直接意志的化身。他们将自己建构成一个“主权者-机构”，其决断是让人民的真实意愿直接在场。解散立宪会议这一事件代表的是理性（议会）与同一性的意愿（自命的主权者）之争。立宪会议代表着碎片化的、需要时间讨论的民意；而列宁主义政党-国家则宣称，历史的真理和人民的根本意愿此刻就在我们这里，无需讨论，只需决断。布尔什维克的这种行为在经典马克思主义那里也是可以找到理论根据的，那就是“虚假意识”这个概念。“虚假意识”是指被统治阶级接受了统治阶级的意识形态，从而无法认识到自己真正的阶级利益。布尔什维克（尤其是列宁）认为广大农民虽然在客观上属于被剥削者，但他们受制于宗教、土地私有观念以及对社会革命党的盲目忠诚。对于选举结果，他们觉得这是广大农民和部分工人阶级在“虚假意识”笼罩下投出的，不能代表他们的长远阶级利益，也不能代表历史发展的必然方向。因此，解散立宪会议不仅不是违反民主，反而是“真理”对“幻觉”的矫正，是保护无产阶级免受自身虚假意识的毒害。既然马克思主义已经“预言”了历史必然走向无产阶级专政，那么任何阻碍这一进程的民主程序（即便它是多数人的选择）都是“反动的”，是虚假意识的残余，必须被武力清除。但是，需要注意的是从传统马克思主义框架看，布尔什维克（列宁）的做法其实有很大问题。列宁搁置了马克思主义的经济决定论。列宁主义的逻辑核心是“真理对幻觉的矫正”。但此时，这个“真理”已不再由历史运动的客观结果裁定，而是由“谁成功了”来裁定。这恰恰是列宁主义对经典马克思主义最彻底、也最具争议的改造。做个比喻就是经典马克思主义以及第二国际那些人认为理论宣传只是催生婆和助产士，孩子必须由无产阶级自身生出。而先锋队逻辑则认为孕妇难产了，靠无产阶级自己永远生不出孩子，必须剖腹产。如果我们脱离马克思主义的框架，就能看出来解散立宪会议其实非常接近马基雅维利主义。列宁主义的特点就是历史唯物主义的“科学”外壳下的“唯意志论”，而这正是列宁对马克思主义最具“马基雅维利色彩”的改造。马基雅维利主义中有两个重要的概念，机运和德性。​机运并非简单的运气，而是指政治生活中那些人力无法控制的、偶然的、非理性的客观环境或历史背景。德性也不是指基督教传统中的诚实、仁慈或谦卑，而是指政治家或君主为了达到目的所表现出的远见、勇气、决断力和卓越才干。其中最核心的德性是“变通”。马基雅维利认为，如果一个人始终保持一种性格，而时代背景（机运）变了，他就会失败。真正的德性是能够根据环境的冷热，随时切换自己的行事风格。而德性的最终目的是建立稳定的国家秩序。在这个过程中，即便采取一些在传统道德看来是“恶”的手段，只要它是有效的、能控制局面的，就属于德性的范畴。​在马基雅维利的语境下，政治不是对既有规则的服从，而是对“时机”的捕捉。马基雅维利主义的真谛在于：平庸的统治者等待机运，卓越的君主创造机运。对于列宁而言，1917年的俄国正处于一个巨大的“法律真空”和“权力瓦解”状态。如果等待立宪会议通过法律程序赋予布尔什维克权力，那么革命的动能就会在无休止的辩论中耗尽。布尔什维克动用武力解散议会，本质上是运用“德性”（政治意志与力量）去强行重塑“机运”。一旦旧的议会消失，苏维埃政权成为唯一的权力实体，群众就会逐渐接受这个“既定事实”。现实的唯一性本身就是一种最强有力的说服。真理不再是书本上的教条，而是来源于行动的成败。如果你成功了，就证明你顺应了“历史趋势”；如果你失败了，那你就是冒险主义。成功不仅赋予了正当性，还追认了你的行动就是“科学真理”。

不过这种决断哲学也造成了一种严重的政治后果。当“成功即合法”成为最高逻辑时，政治就演变成了纯粹的力量博弈。在这种暴力虽然是“创始性”的，但由于它缺乏一个超越性的道德或契约约束，它必须不断地通过新的暴力来维持。这也就是为什么在解散立宪会议后，布尔什维克必须进入一个长期的军事共产主义和内战动员状态。“唯有成功者才能定义什么是真理”的决断哲学，本质上弃绝了任何超越于行动的规范性评价标准。它让政治变成了一个密封的、自我循环的系统。当未来被彻底开放给力量的赌博，任何反对者的质疑都可以被同一套逻辑驳回：“你之所以质疑我，只因为你还没赢。”与之对应的就是，当他们输了之后，别人也可以同样嘲讽回去。
