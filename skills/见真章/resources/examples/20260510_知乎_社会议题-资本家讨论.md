# 怎么看待“你不是讨厌资本家，而是恨自己不是资本家”这句话？

因为“骂资本家”很大程度上是既有权力结构再生产的一部分。

这样的循环可以上溯几千年。成百上千农民起义后为什么封建王朝依旧？

因为农民起义的目的是为了当皇帝。

革命不彻底就是彻底不革命，如果只是在暴力手段下分了田，而没有一个相匹配的制度维持新建的权力结构，那么迟早天火会烧了黄册鱼鳞册，土地兼并会再度开始。

“骂资本家”的合法性来自何处？

第一，互联网去中心化的场域下对“精英”阶层的权力优势；第二，主流意识形态下的道德优势。

很不幸的是，这两点优势仅仅建立在“资本家-平民”的权力结构上，对这一结构的“形式颠覆”除了能让自己的话语在特定场合获得支持和合法性，别无他用。

为什么说是“形式颠覆”？

因为批判的对象错了。

我始终坚持一个观点，个体意志是无法对抗制度意志的。在这一情境下，对“资本家”的批判，指向的是个体，实际上无法构成对所谓“资本主义”制度的颠覆。

而所谓“恨自己不是资本家”，本质上是将个体作用放大到决定性的位置了。

这个道理大家在封建王朝的时候都懂——不能期盼所有皇帝都如圣人一般爱民如子，这是封建制度的局限性。到了资本主义阶段，却又想着只要我是大资本家，我便造福人类，移居火星。

所以这个问题虽然情绪化了，但结果客观来说，都是一样的。

也就是说，你只是批倒了董事席位上的资本家，又不是拆了整条华尔街。

就算你拥有海瑞一般圣心，坚决要当一个平头老百姓，也怕是上一任带有董事长屁股印的真皮座椅余温尚未散去，便有人抢下这块权力真空了。
