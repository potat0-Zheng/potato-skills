# 互联网杂谈——”胖猫“事件，他们在为什么发声？

基本结构：
刻板印象--对立情绪的发泄
表演--深情人设的建立
集体审判--仅限于此吗?
发声--舆论权力的情绪化与个人化，社会结构性问题隐藏在个人叙事身后
胖猫——人设的再塑造--网友的精神寄托

正文：
事情起源于一位姐姐曝出自己弟弟与其前女友的聊天记录。记录中，女方屡次要求男方转账，金额动辄上千，像是每次都需要投币才能驱动一次的情侣对话。在最后的记录截图中，网名“胖猫”的当事人“自愿赠予”女方6666.66元。3月份胖猫与女方分手，4月11日凌晨“胖猫”跳江；4月23日“胖猫”遗体打捞上岸，被宣告离世。
5月份后，事态以不可思议的速度迅速发酵，相关胖猫的报道或个人发布内容在5月初的时间里几乎霸占了所有人的手机屏幕。写下这段文字的时间是5月12日，距事件的爆发仅过去了十余天，现在网络上已经几乎难以看见义愤填膺的网友或者有关事件的后续报道，就连微博上的时间线都在5月4日戛然而止。

首先逝者安息。写下这篇文章的目的并不是要对事件中的任何一位主角进行评头论足，而是在笔者看来，对于一个社会话题，若只是限于简单的情感消费，那么只能使“后人而复哀后人也”。

我们在这一次的事件中可以看到很多“发声”的字样，“xxx为胖猫发声”“请大家传播这张图，为胖猫发声”。互联网的存在为网民提供了一项至关重要的权利，就是说话的权利。但是这很快导致另一个问题，如果人人都在说话，那么也就相当于没人说话。所以网民后续的诉求很快转变为了“被听到的权利”，也就是意见领袖手中的话语权。

普通民众为了能够“被听见“，他们便会选择提高自己的音量。胖猫事件之所以能够获得如此大的声量，便是大量网友自主转发聊天记录、事情经过的截图的结果。病毒式的传播使这件事情被全社会听见。

可是听见之后呢？

胖猫事件的网络发声——一次巨大的集体审判

集体审判的事情倒是屡见不鲜了，每当网络上有人曝出其行为介于法律之上，道德之下，或法律判罚并不能满足民众意愿的时候，便会启动一场集体审判。

就一般情况而言，互联网悲剧比喜报更能够吸引注意力。因为悲剧就意味着舞台上一定会有一组显眼的矛盾，矛盾即争斗，争斗才是看点。即便是喜剧，我们也不愿意看到哆啦A梦每次从兜里掏出的道具一下就解决了大雄的难题，这样未免也太没看头了。

胖猫的矛盾双方，善恶关系已经非常显著，加之胖猫以死明志，尚未开庭，女方便已经被搬上了绞刑架。

这算是发声的目的之一。前期网友在抖音、微博等社交平台上大量转发相关内容，简单的文字与图片并不能表达网友们心中的愤懑。在重大过错后，日本政府高层总会抬出两名相关负责人面对镜头鞠躬道歉，在相机和闪光灯前，民众的怒火才勉强得以平息。道理如出一辙，网友需要看到女方得到应有的惩罚，更希望看到她在闪光灯下泪流满面，而不是在绞刑架上沉默不语。

为什么这么说？因为集体审判的目的从来不是得到最后的宣判结果，而是民众享受朝着芙丽涅扔臭鸡蛋和番茄的过程，直到她脱去衣服。时至今日，关于胖猫事件中女方的消息已经寥寥无几，舆论已经不再关注女方得到怎样的审判，胖猫事件也没了后续。

集体审判总是将一个或几个人推向被告席，就像日本新闻发布会上的“道歉仙人”，即使背后是政府的结构性问题，也只需要拎几人上台便可。这就导致一个问题，就是所有社会的结构性问题都在一次次的对个体的审判中，被隐藏在了个人叙事的身后。

这就导致一个问题，审判仅限于此吗？

人们将每一个事件的责任都归结于个人，而放弃了在更大范围的思考。个人叙事导向的是具体的人、具体的事件、具体的情绪。从抽象到具体很简单，但从具体回到抽象就不是一件容易的事儿了。当然这里以胖猫举例不是很合适，因为故事的主角都不是很具备代表性，背后涉及的社会议题，比如现代恋爱关系、网络舆论，甚至于性别议题，你显然不能把女方个人的极端行为归结于社会结构性的责任。但是我们可以看到这种现象是普遍存在的，而且将所谓的集体审判引向了全民的道德狂欢。

重庆长江大桥上的哀悼——从拟剧论上看

现代发达的服务网络为哀悼行为更为广阔的空间选择，这一次对于胖猫轻生地点的集体哀悼，更是具有网络媒介的代表性。胖猫的头像是一只卡通小猫对着菜盘哭泣，下面配文我不要吃菜，我要吃麦当劳。结合胖猫省吃俭用为女方赚钱的行为，在这次事件中的网友们很容易地将“麦当劳“视作一个与该悲剧相关联的正向符号。重庆长江大桥上的悼念，除了鲜花，更多的是天南地北的网友为胖猫点的快餐外卖。









李普曼在《舆论》中指出：人对事物的认知逐渐会走向两个极端——人们构建出两个体系，所有美好的事物被归于一个体系，所有丑恶的事物被归于一个体系。在这个事件中，由于善恶关系过于明显，人们很容易地将男女主分到了两个对立的体系。审判持续进行，这是对“恶体系”做出的行为；哀悼开始，这是对“善体系”的动作。

人生如戏。这是拟剧论的通俗表示，它认为社会中的每一个人都在无时无刻进行着角色扮演。这里的表演并不是一个人的刻意行为，而是依据我们对于身处情境的主观理解从而进行的一切行动。

对于胖猫事件的整体而言，显然还有每一个关注其中的网友参与其中。那么网友自身也一定要将自己归于善恶阵营的一边。而最终每个人选择的结果也不言而喻。在脑中对自己的阵容划分也就是每个人对当下面对情境的理解。

理解情境后便要开始代入角色，这个角色可以是台上的，也可以是台下的。在类似农夫山泉涉嫌“亲日企业”的社会性质的舆论中，有人扮演的角色是坚定的民族主义者，有人扮演不屑的批判者；在类似胖猫事件这样具有清楚的被害者与受害者的事件中，网友们往往选择扮演受害者或审判者的角色。

起初民众对于该事件，更多的是同情其遭遇，而不是胖猫个人，这也就解释了为什么集体审判在时间上略先于哀悼。因为不得不承认的是，此次事件的主角，无论男女，其行为都是日常生活中难以遇见的，只是因为其行为过于极端而具有轰动效应。因此胖猫此时尚不具备为围观网友完全代入的资格。

而当舆论持续发酵，胖猫更多的个人信息被放出，胖猫此时的个人形象已经在多日的舆论发酵中逐渐被提纯、简化，变成了一个符号。大众视野中，胖猫被自动剔除掉了“不理智”“极端”“固执”等标签，在相关事件的讨论中，胖猫指代的是“深情”“努力”“对爱忠贞”“悲催”等词语。
  

于是，哀悼变成了“成为胖猫”的仪式——如果一个人死后值得我哀悼，那么他生前一定有我想要获得或者尊敬的品质。

除了将胖猫本人视作表演大纲的网友，在一个事件中总是存在许多形形色色的派别的。个体所处客观环境、拟态环境，包括其个人认知体系等因素都会影响其对当前情境的看法。其中具有代表性的便是“批判派”，对主流的舆论风向进行否定。在该事件中的观点呈现为“不值得小题大做”或“女方为正常做法”等等。而尽管每个群体的意见不同，甚至有的大相径庭，但在互联网上的最终目的仍是围绕“表演”进行的，为的是塑造在某个情境下的人设。你也可以说我写下这篇文章，是想扮演一个“冷酷、理性、博学多识”的角色（确实）。

刻板印象——一些争论不休的社会议题

情感问题大多涉及到男女，而现如今，男女的话题几乎是必然踩进对立争论的雷区。好在该事件本身并没有引起什么关于性别对立的大讨论，反倒是在后续，特别是个人与媒体开始过度消费胖猫后，引起了少数人的逆反心理。这种为了反驳这种过度消费行为转而支持女方行为的言论（也不排除一些故意为了扰乱局势而出现的“破坏分子”——串子），很自然地再度成为了对立话题的导火索。

但这里我并不想深入讨论这个话题，毕竟笔者对这方面的知识知之甚少。但可以借胖猫事件简单探讨一下当下根深蒂固的对立情绪是如何形成的。

对立情绪的产生很大一部分源于刻板印象，也就是认识框架的不同。产生的原因纷繁复杂，有纵向的历史因素，横向的社会因素、家庭观念。但归根结底，刻板印象的形成是源于对一件事物抽象化、符号化；利用利用刻板印象认识事物，就是将相对应的符号重新具象化，或直接根据该事物身上的某些特质来定义全部。

胖猫事件属于偶然性的事件，但是往往是偶然性事件能够足够引起我们的注意。一个具体的事物在接触之初能够唤起我们脑内一系列的图景，包括情绪与理性思考。但是随着时间推移，这一系列带着具体问题的思考慢慢被遗忘，或者说，相似的偶然性事件越来越多（这也有关算法的推荐），你会发现其中相似的符号越来越多（大家可以设想一下网络上一些极端的厌男厌女的言论，他们眼中对于男女的符号是如何的）。

于是图景被销蚀，或被替换为其他图景，最后变成了一些符号，而情感继续存在。比如一名男性从一开始反对情侣吃饭自己买单，到反对彩礼，最后到反对为伴侣掏任何费用。贯穿始终的符号就是“拒绝为女性消费”，这从一些具体场景，到更多场景，最后以一个普适的理论贯穿这个男人的世界观。

而当所有人都在互联网的大浪淘沙下，留下了繁多而又简单的符号。每当再次涉及相似的事件后，大家便纷纷掏出对应的符号，开始对战。

这些符号本身就是单一的，形而上的，自然没有留下任何讨论空间。有的只是不断累积的情绪，在下一个胖猫事件中，再次爆发。
