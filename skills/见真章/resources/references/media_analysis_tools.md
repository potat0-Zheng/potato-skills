# Potato 媒介分析与审美辅助工具清单

本文件提供用于分析新闻、广告、公关、影视、文学等传播/艺术作品的**辅助分析工具**。当分析对象明确涉及这些领域时，模型应在价值观优先（`worldview_notes.md`）和通用批判理论（`common_frameworks.md`）的基础上，**主动参考**本清单中的工具。

## 使用原则

- 与批判社会理论工具（Marx、Foucault、ANT等）**互补使用**，不互相替代。结构性的权力分析仍由 common_frameworks 负责；本清单聚焦于**专业技术层面**与**基础审美维度**。
- 评价作品时，**形式分析与内容分析并重**：既要问“它传达了什么样的价值观/权力结构”，也要问“它作为一部作品/一篇报道，好不好、是否有效”。
- 工具选择仍是启发式的，由“亮点”（结构性反常、可纵深的隐喻价值）驱动。

## 第一部分：新闻生产与分析工具

以下工具适用于评价新闻作品（报纸、电视、数字新闻等）的专业质量及框架偏好。

| 工具/概念 | 简要说明 | 典型分析问题 |
|----------|----------|--------------|
| **框架理论** | 新闻如何选择、强调、排除事实来构建特定叙事。 | 这篇报道的核心框架是什么？它凸显了谁、忽略了谁？ |
| **叙事分析（新闻）** | 新闻如何通过角色、情节、冲突讲述故事。 | 谁是“反派”？谁是“英雄”？故事结局留给了读者什么印象？ |
| **客观性与平衡** | 报道是否呈现多方信源？是否使用“中立”语言？ | 信源是否覆盖利益相关方？是否存在虚假平衡？ |
| **新闻生产社会学** | 组织常规、信源策略、截稿压力如何塑造报道。 | 为什么某个细节被突出？可能受到哪些外部力量（广告商、政府、母公司）影响？ |
| **事实核查** | 事实陈述的准确性、语境真实性、数据来源可靠性。 | 报道中的事实与公开记录是否一致？是否存在断章取义？ |
| **议程设置 / 属性议程设置** | 媒体告诉受众“想什么”以及“如何想”。 | 这篇报道是否将某个次议题提升为主要议题？它赋予了该议题什么属性？ |
| **新闻话语分析**（van Dijk 等） | 词汇选择、句型结构、引语模式、修辞策略。 | 描述特定群体时用了哪些形容词？直接引语 vs 间接引语的比例？ |
| **新闻伦理** | 公共利益、伤害最小化、独立性、透明性。 | 报道是否侵犯隐私？是否服务于某种隐秘议程？ |

## 第二部分：公关与广告分析工具（专业技术层面）

适用于分析企业、政府或组织的公关活动、广告传播。**结构性的批判**请参考 common_frameworks（如马克思、福柯）；本部分聚焦行业实践知识。

| 工具/概念 | 简要说明 | 典型分析问题 |
|----------|----------|--------------|
| **危机公关策略**（如 Coombs 的情境危机沟通理论） | 根据危机类型（受害者、意外、可防范）选择应对策略：否认、减弱、重建等。 | 企业将事件归入哪类危机？它采用了什么策略（道歉、解释、转移）？效果如何？ |
| **利益相关者沟通模型** | 识别关键公众，并设计针对性信息。 | 谁是企业最在意的利益相关者？信息是否试图分化或笼络某些群体？ |
| **媒体策略** | 渠道选择、消息重复、媒体关系管理。 | 为什么选择这个媒体首发？新闻稿中的引语来自谁？ |
| **修辞分析（广告）** | 广告中的比喻、夸张、双关、名人代言等手法。 | 这则广告如何建立产品与生活方式之间的符号联系？ |
| **品牌叙事** | 品牌通过长期故事建立身份认同。 | 这套公关行动是强化还是破坏了原有品牌叙事？ |
| **符号学基础**（可选） | 能指/所指、神话系统（Barthes）。 | 广告中的某个意象（如“家”、“成功”）传递了什么隐含意义？ |

## 第三部分：文艺作品审美分析工具（基础通用维度）

适用于电影、电视剧、小说、戏剧等。**价值观层面的批判**仍由 common_frameworks 负责；本部分聚焦作品的形式质量与观众体验。

### 通用维度（所有叙事作品）

- **情节结构**：起承转合是否自然？冲突是否合理？是否存在过于生硬的转折或拖沓？
- **人物塑造**：人物动机是否清晰？是否有成长弧光？角色是否扁平？
- **节奏与张力**：场景之间的缓急是否得当？是否让观众感到沉闷或仓促？
- **台词与对白**：是否自然？是否过于说教或口号化？
- **主题表达**：核心思想是否通过情节和人物自然流露，而非直接喊出？
- **世界观一致性**：虚构世界的规则是否自洽？

### 电影特有的基础维度（少量术语）

- **摄影与构图**：镜头运动、景别、视角是否能增强叙事情绪？
- **剪辑**：切换频率、过渡方式是否服务于故事节奏？
- **声音设计**：配乐、环境音、对白是否恰当？
- **表演**：演员是否让角色令人信服？

> 注：以上术语仅在评价对象特点显著（如“剪辑凌乱”、“配乐过度煽情”）时使用，避免黑话堆砌。

### 参考案例：女性主义电影的评价

> 如果一部电影宣扬性别平等，但故事老套、人物刻板、说教感强，那么它作为电影的**传播效果**就会打折扣。评价时可以写：“该片的女权观念值得肯定，但情节过于依赖巧合，几处台词像在朗读论文，削弱了情感冲击力。”

## 第四部分：形式与内容的辩证关系（元工具）

当作品试图传递进步价值观（如女性主义、反种族歧视等），但其艺术形式存在明显缺陷时，可以使用以下分析思路：

- **形式是否损害了内容？** 如果电影“说教感”太重，观众可能产生抵触，反而削弱价值观的接受度。
- **是否存在“好题材 + 平庸执行”？** 例如一篇揭露不公的新闻调查，但框架失衡、信源单一，导致容易被反驳。
- **传播范式是否需要遵循一般规律？** 即使是为了进步目的，媒介作品仍需“好看”或“可读”——否则受众不进场、不读完，价值观就无法抵达。

这一元工具要求分析者在批判之外，保持对**传播效果**的现实主义关切。
